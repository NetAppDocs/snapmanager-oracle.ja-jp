= セカンダリストレージから保護されたバックアップをリストアする
:icons: font
:imagesdir: ../media/


[role="lead"]
保護されているバックアップはセカンダリストレージからリストアできます。ただし、プライマリストレージにバックアップが存在する場合、セカンダリストレージからバックアップをリストアすることはできません。
