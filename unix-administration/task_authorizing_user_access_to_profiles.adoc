= プロファイルへのアクセスの許可
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager では、ロールベースアクセス制御（ RBAC ）に加えて、プロファイルにパスワードを設定して不正アクセスを防止することができます。

. 次のコマンドを入力します。 'mo credential set -profile -name profile_name [-password password]'
