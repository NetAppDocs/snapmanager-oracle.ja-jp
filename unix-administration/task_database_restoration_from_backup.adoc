= バックアップからのデータベースリストア
:icons: font
:imagesdir: ../media/


[role="lead"]
給与計算データベースのアクティブなコンテンツが誤って失われたり破壊されたりした場合、 SnapManager と NetApp Management Console のデータ保護機能を使用すると、ローカルバックアップまたはセカンダリストレージからデータをリストアできます。
