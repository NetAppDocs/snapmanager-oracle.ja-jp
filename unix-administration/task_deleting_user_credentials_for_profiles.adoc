= プロファイルのユーザクレデンシャルを削除する
:icons: font
:imagesdir: ../media/


[role="lead"]
プロファイルのユーザクレデンシャルを削除して、ユーザがアクセスできないようにすることができます。

. ユーザのプロファイルクレデンシャルを削除するには、次のコマンドを入力します。 'mo credential delete -profile -NameProfile_name'
