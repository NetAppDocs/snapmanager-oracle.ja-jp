= バックアップを無期限に保持
:icons: font
:imagesdir: ../media/


[role="lead"]
バックアップを無期限に保持するには、保持ポリシーの削除対象外にするように指定します。

. バックアップを無制限に保持するように指定するには、次のコマンドを入力します。 `mo backup update -profile profile_name {-labellabel[data|-archives]|-idID}-retain-unlimited`
