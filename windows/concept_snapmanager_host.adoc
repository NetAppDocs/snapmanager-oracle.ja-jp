= SnapManager ホスト
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager ホストとは Windows サーバであり、ほかのネットアップ製品も稼働します。

SnapManager ホストには次の製品がインストールされます。

* SnapDrive for Windows の略
* Host Utilities のことです


SnapManager ホストはサービスとして実行されます。
