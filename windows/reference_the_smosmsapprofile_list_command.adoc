= smo profile list コマンド
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
このコマンドは、現在のプロファイルのリストを表示します。



== 構文

[listing]
----

        smo profile list
\[-quiet \| -verbose\]
----


== パラメータ

* * - Quiet *
+
コンソールにエラーメッセージのみを表示します。デフォルトでは、エラーおよび警告メッセージが表示されます。

* * -verbose *
+
エラー、警告、および情報メッセージがコンソールに表示されます。
