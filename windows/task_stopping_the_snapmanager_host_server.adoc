= SnapManager ホストサーバを停止します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager の使用が終了したら、必要に応じてサーバを停止できます。

. サーバを停止するには、 root ユーザとして次のコマンドを入力します。 smo _server stop
