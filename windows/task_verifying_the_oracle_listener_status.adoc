= Oracle リスナーのステータスを確認する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Oracle リスナーのステータスを確認するには、 lsnrctl status コマンドを使用します。

* データベースに接続する必要があります。


標準の Oracle インストールでは、データベースのリスナー・ポートが 1521 に設定されます。

. コマンドプロンプトで、 lsnrctl status と入力します
