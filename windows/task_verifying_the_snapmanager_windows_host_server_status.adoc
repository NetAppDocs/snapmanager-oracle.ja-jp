= SnapManager ホストサーバのステータスの確認
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
コマンドを実行したり、 SnapManager 処理を開始したりするには、サーバが実行されている必要があります。処理を実行する前に、サーバのステータスを確認する必要があります。

. [ サービス ] ウィンドウで 'Oracle の SnapManager 3.3 を選択します
. ステータス列でステータスを確認します。
